# Cosmic Factions

A simple factions plugin to add in to your minecraft be pocketmine server. With this plugin players can now create and battle against other factions! Bringing back the real factions!

# Features

- [x] Manage factions
- [ ] Money and Power
- [ ] Claiming
- [ ] Faction Chat
- [ ] Faction Home
- [ ] War/Outpost
